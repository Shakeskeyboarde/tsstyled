Interface IStyledTemplate<TStatic, TOuterProps, TInnerProps>

Styled template function with static utility methods for modifying the styled component's prop values and replacing the default component props type.

    Transform all props based on previous props. The new props must still be compatible with the wrapped component's props.

    This method removes props that are not returned.

    • Try the use method if you want to provide default prop values.
    • Try the set method if you want to set (not remove) some prop values.

  • Inherited from IStyledTemplateMod.set

    Partially set props. This is effectively a "merge" with undefined value skipping.

    Be careful that you do not unconditionally overwrite props passed in to the styled component!

    • Try the use method if you want to provide default prop values (when the current value is undefined).
    • Try the map method if you want to set and remove values.
